# いま

DMM.com で働くエンジニアです。

現在、仕事では <br>

- Go を使ったバックエンド開発
- React を使ったフロントエンド開発
- CloudFormation を使ったインフラ構築

をやっています。

<br>

バックエンド開発がメインです。<br>
アプリケーションレイヤの設計について考えるのが好きです。

# 余談：学生時代の研究

主に無線センサネットワークに関する研究をしていました。

研究を通して、低レイヤの知識や <br>
リソースが少ないハードウェアに対する効率的なプログラムの書き方などを学びました。

触っていた技術は TinyOS や nesC などです。
